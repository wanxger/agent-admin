# Agent Admin

Agent Admin 是一个基于 [Agent Client Protocol (ACP)](https://agentclientprotocol.com/) 的命令行工具，旨在简化与 AI 代理的交互过程。

## 简介

Agent Admin 提供了一个统一的接口来管理多个 AI 代理任务，支持：

- 通过配置文件批量执行任务
- 自定义工作目录
- 灵活的代理命令配置
- 完整的测试支持

## 特性

- **配置文件驱动**：使用 YAML 配置文件定义任务和参数
- **多任务支持**：一次配置可以执行多个任务
- **灵活的工作目录管理**：支持项目和任务级别的工作目录设置
- **优先级配置**：命令行参数优先级高于配置文件
- **内置测试代理**：提供测试代理用于集成测试

## 架构

Agent Admin 是一个 Rush 管理的 monorepo 项目，包含以下主要部分：

- `apps/agent-admin/` - 主应用
- `libs/prompt/` - 可复用的 ACP 客户端库
- `apps/website/` - 文档网站
